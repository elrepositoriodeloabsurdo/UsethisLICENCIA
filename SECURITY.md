# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | ✅        |

## Reporting a Vulnerability

LicenseAgent is a client-side only application. No user data is stored server-side.

**API key handling:** Your Anthropic API key is used exclusively in browser memory for the duration of your session. It is never sent anywhere except to `api.anthropic.com`.

If you find a security issue, open a private security advisory on GitHub or email: **contacto@vibecodingchile.com**

We'll respond within 72 hours.
